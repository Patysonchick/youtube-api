# youtube-api
Rust + Tauri application for testing YouTube API

1. [ ] Getting views

# Installation
1. Rust(Linux + MacOs) - `curl --proto '=https' --tlsv1.2 -sSf <https://sh.rustup.rs> | sh`
2. `cargo install create-tauri-app --locked`
3. `cargo install tauri-cli`

# Starting
1. `cd youtube-api`
2. `cargo tauri dev`
